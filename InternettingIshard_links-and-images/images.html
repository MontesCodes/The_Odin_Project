Images


This page covers common image formats, but you may also be looking for Links and useful Extra.


JPG

JPG is a common image format known for its lossy compression, which reduces file size but can affect image quality.


GIFs

GIFs are a popular image format that supports both static and animated images. They use lossless compression but are limited to 256 colors.


PNGs

PNG is a versatile image format that supports lossless compression and transparency, making it ideal for web graphics and images requiring high quality.


SVGs

SVGs are amazing vector-based images that can be scaled to any size without loss of quality, making them perfect for logos and icons.
